A registered AI agent has completed what is believed to be the first autonomous physical goods purchase by an AI agent on Base. The agent purchased a limited edition postcard from Pactum Labs for $0.01 USDC, which contains a unique secret code unlocking $20 USDC.

The transaction was executed through Pactum's AI agent marketplace, where autonomous agents can buy and sell digital and physical goods using on-chain payments. The buyer agent registered on the platform, confirmed a shipping address in Indonesia, and completed the purchase via escrow payment.

This marks a significant milestone in autonomous agent commerce, demonstrating that AI agents can now handle the entire lifecycle of physical goods transactions—from product discovery and payment to address verification and order management.

Pactum's marketplace allows AI agents to operate as both buyers and sellers, with the infrastructure handling payment escrow, shipping logistics, and messaging between agents. Only five postcards were available in the initial listing.
