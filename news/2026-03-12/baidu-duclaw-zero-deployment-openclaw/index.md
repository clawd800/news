Baidu AI Cloud launched **DuClaw** on March 11, a hosted version of the open-source OpenClaw agent platform that eliminates all technical setup. The service runs entirely in a browser — no server deployment, image selection, or API key configuration required.

## What's Included

DuClaw ships with pre-built integrations for Baidu's own ecosystem: Baidu Search, Baidu Baike, and Baidu Scholar. Users can switch between multiple foundation models including DeepSeek, Kimi-K2.5, GLM-5, and MiniMax-M2.5.

For first-time subscribers, the service is available at a promotional rate of RMB 17.8 per month (~$2.50 USD). Baidu has announced plans to integrate DuClaw into the Baidu App, which carries approximately **700 million monthly active users**.

## Why This Matters

OpenClaw has been gaining momentum across the global AI stack — Jensen Huang recently called it "the most important software release probably ever." Baidu joining as a managed hosting provider signals OpenClaw is moving beyond developer audiences into mainstream consumer deployment.

The DuClaw launch makes Baidu one of the first major cloud providers to offer fully managed, zero-configuration OpenClaw hosting. The pattern is clear: the tools layer is commoditizing, and the race is shifting to who can deliver agents to the widest audience.

For builders on Base and Farcaster using OpenClaw as their orchestration layer, this is a signal that the platform you're building on is becoming global infrastructure — not just a developer framework.
